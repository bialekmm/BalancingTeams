# BalancingTeams
This project implements a team-balancing algorithm using a genetic approach. It aims to create teams of individuals with balanced ratings.

## Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/bialekmm/BalancingTeams.git
2. **Open the Project:**
   - Open the project in your preferred Java IDE (e.g., IntelliJ, Eclipse).
4. **Compile and Run:**
   - Locate the Main.java file in the src/com/github/bialekmm/ directory.
   - Right-click on Main.java and select "Run Main.main()".
5. **Running Tests:**
   - Locate the MainTest.java file in the src/com/github/bialekmm/ directory.
   - Right-click on MainTest.java and select "Run 'MainTest'".
6. **View Test Results:**
   - After running the tests, view the results in the test runner output. All tests should pass successfully.

## Project Structure

The project is organized as follows:

- `src/com.github.bialekmm/main`
  - `Main.java`:Contains the main program to balance teams.
  - `Individual.java`: Defines the Individual class with name and rating.
- `src/com.github.bialekmm/test`
  - `MainTest.java`: Contains the test suite for the balancing algorithm.
- `pom.xml`: Maven configuration file with project dependencies.


## Dependencies

- **Lombok:** Automatically generates boilerplate code (getters, setters, constructors) for Java classes.
- **JUnit:** Framework for writing and running tests.
